Real-time 6-DoF tracking system using Azure Kinect RGB-D data and ROS2 Humble. A custom YOLOv8 model segments the oil pan, depth masking builds the object point cloud, and pose is estimated with RANSAC coarse alignment plus Wunsch refinement against a CAD model. The resulting transform drives xArm inverse kinematics for continuous object-centered tracking.
